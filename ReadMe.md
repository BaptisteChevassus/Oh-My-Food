# Ohmyfood 🍴
Ohmyfood est une jeune startup qui souhaite s'imposer sur le marché de la restauration. Ce projet consiste à développer un site "mobile first" répertoriant les menus de restaurants gastronomiques, permettant aux clients de composer leur repas à l'avance pour réduire les temps d'attente.

# 📋 Présentation du Projet
Ce projet a été réalisé dans le cadre de la formation Développeur Web chez OpenClassrooms. L'objectif principal est de transformer des maquettes complexes en un site fonctionnel et dynamique, uniquement en HTML et CSS/SASS.

# 🎯 Objectifs Techniques
Approche Mobile First : Développement pensé pour mobile, puis adapté pour tablette et desktop.

Intégration SASS : Utilisation de SASS pour une structure de code CSS modulaire et maintenable.

Animations CSS : Mise en place d'animations sophistiquées (sans JavaScript) pour enrichir l'expérience utilisateur.

# ✨ Fonctionnalités & Animations
Le site intègre des interactions fluides demandées par le CTO :

Loading Spinner : Une animation d'introduction lors du chargement de la page d'accueil.

Effet d'apparition : Affichage progressif des plats sur les pages de menus.

Interactions au survol :

Cocher un plat lors de la sélection.

Animation de remplissage progressif des icônes "cœur" (favoris).

Effet d'ombre et d'éclaircissement sur les boutons.

# 🛠️ Technologies
HTML5

SASS / CSS3 (Utilisation de variables, mixins et architecture modulaire)

Git / GitHub (Gestion de version)
